Lista Ordered

The ol element allows you to define ordered lists or bullets (“Ordered List”), either with numbering or alphabetically.
its labels are: <ol> </ol>


Exemple:

the code to make an ordered list is:

<ol>
<il>Point one </il>
<il>Point two </il>
<il>Point three </il>
</ol>

And this is how it would look on the web:

  1. Point one
  2. Point two
  3. Point three

Unordered list

ul from "unordered list" -unordered list. creates an unordered list.
its labels are: <ul> </ul>


Exemple:

the code to make an unordered list is:

<ul>
<il>Point one </il>
<il>Point two </il>
<il>Point three </il>
</ul>

And this is how it would look on the web:

  • Point one
  • Point two
  • Point three

List element

The li element of the English item list or list element declares each of the elements of a list.
its labels are: <il> </il>


Exemple:

The code to make an il is first making an ul or ol and then putting the il for each element within the list:

<ul>
<il>Point one </il>
<il>Point two </il>
<il>Point three </il>
</ul>

And this is how it would look on the web:

  • Point one
  • Point two
  • Point three
