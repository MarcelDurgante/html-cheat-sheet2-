Lista Ordenada

El elemento ol permite definir listas o viñetas ordenadas (“Ordered List”), bien con numeración o alfabéticamente.
Sus etiquetas son: <ol> </ol>


Ejemplo:

el codigo para hacer una lista ordenada es :

<ol>
<il>Punto uno </il>
<il>Punto dos </il>
<il>Punto tres </il>
</ol>

I asi es como quedaria en la web :

  1. punto uno
  2. punto dos
  3. punto tres

elemento de lista

El elemento li del ingles item list o elemento de lista declara cada uno de los elementos de una lista.
Sus etiquetas son: <il> </il>


Ejemplo:

el codigo para hacer un il es primero haciendo un ul o ol para despues poner el il por cada elemento dentro de la lista :

<ul>
<il>Punto uno </il>
<il>Punto dos </il>
<il>Punto tres </il>
</ul>

I asi es como quedaria en la web :

  • punto uno
  • punto dos
  • punto tres
