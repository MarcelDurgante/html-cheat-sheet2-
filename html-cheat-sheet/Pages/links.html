Link

El Elemento HTML Anchor <a> crea un enlace a otras páginas de internet, archivos o ubicaciones dentro de la misma página, direcciones de correo, o cualquier otra URL.






































Ejemplo:

Con este codigo haces un link a una pagina externa en una nueva pestaña

<a href="https://www.google.com/" target="_blank"> Google </a>
Google

con este otro codigo hacemos un link a otra pagina de la misma web

<a href="#Titulo"> Subir pagina </a>
Subir pagina
