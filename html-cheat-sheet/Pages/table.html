Table

The HTML Table Element represents data in two or more dimensions.
its labels are: <table> </table>


Exemple:

codigo:


    <Table>
        <tr>
            <th>First name </th>
            <th>Last name </th>
        </tr>
        <tr>
            <td>John </td>
            <td>Doe </td>
        </tr>
        <tr>
            <td>Jane </td>
            <td>Doe</td>
        </tr>
    </Table>

As it looks on the page

First name Last name
John Doe
Jane Doe

Fila tabla

The HTML element table row (table row) <tr> defines a row of cells in a table. These can be a mix of <td> elements. and <th>
its labels are: <tr> </tr>


Exemple:
First name Last name
John Doe
Jane Doe

Table header

The HTML element defines a cell as the heading for a group of cells in a table. The exact nature of this group is defined by the scope and headers attributes
its labels are: <th> </th>


Exemple:
First name Last name
John Doe
Jane Doe

table cells

The HTML element Table cell td defines the cell of a table that contains data. This participates in the table model.
its labels are: <td> </td>


Exemple:
First name Last name
John Doe
Jane Doe
