Images

img

El elemento de imagen HTML <img> representa una imagen en el documento.


Ejemplo:

este es el codigo utilizado para poner la imagen siguiente :

<img src="http://placekitten.com/200/200"" alt="Gatos bonitos" width="180px" height="180px" >


En esta imagen se rodea con la etiqueta <a> enlazando con la paguina web de los gatos
