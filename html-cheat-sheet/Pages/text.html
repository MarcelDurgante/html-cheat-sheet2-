Textos

Strong

El elemento strong es el apropiado para marcar con especial énfasis las partes más importantes de un texto.
Sus etiquetas son: <strong> </strong>



Ejemplo:

El dinero es importante pero la salud lo es más.

Strikethrough

El elemento HTML coloca un tachado (línea horizontal) sobre el texto.
Sus etiquetas son: <strike> </strike>



Ejemplo:

El dinero es importante pero la salud lo es más.

Italics

El elemento HTML es el apropiado para marcar con énfasis las partes importantes de un texto.
Sus etiquetas son: <em> </em>



Ejemplo:

El dinero es importante pero la salud lo es más.

Subscript

El elemento HTML define un fragmento de texto que se debe mostrar, por razones tipográficas, más bajo, y generalmente más pequeño, que el tramo principal del texto, es decir, en subíndice
Sus etiquetas son: <sub> </sub>



Ejemplo:

El dinero es importante pero la salud lo es más.

Subscript

El elemento HTML define un fragmento de texto que se debe mostrar, por razones tipográficas, más alto, y generalmente más pequeño, que el tramo principal del texto, es decir, en superíndice.
Sus etiquetas son: <sup> </sup>



Ejemplo:

El dinero es importante pero la salud lo es más.
