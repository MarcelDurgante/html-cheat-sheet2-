Aside

The HTML element <aside> represents a section of a page that consists of content that is indirectly related to the main content of the document.


Exemple:
<Aside>
<h2>Header</h2>
<p>Lots of indirectly related content.</p>
</Aside>

Article

The HTML article Element (<article>) represents a self-contained composition in a document, page, an application or on the site, which is intended to be distributed independently or reusable, for example, in the indication.


Exemple:
<Article>
<h2>Google Chrome</h2>
<p>Google Chrome is a web browser developed by Google, released in 2008. Chrome is the most popular web browser in the world today!</p>
</Article>

Nav

The HTML element <nav>represents a section of a page whose purpose is to provide navigation links, either within the current document or to other documents.


Exemple:
<nav class="menu">
<ul>
<li>Inicio</li>
<li>Sobre nosotros</li>
<li>Contacto</li>
</ul>
</nav>

Footer

The HTML element Footer (<footer>) renders a footer for the closest section content or section root element

</footer>
