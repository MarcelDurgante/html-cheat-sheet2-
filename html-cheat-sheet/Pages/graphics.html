SVG

The svg element is a container that defines a new coordinate system and viewport.
It is used as the outermost element of any SVG document, but it can also be used to add an SVG fragment into an SVG or HTML document.


Exemple:
<svg width="150" height="100" viewBox="0 0 3 2">
<rect width="1" height="2" x="0" fill="#008d46" />
<rect width="1" height="2" x="1" fill="#ffffff" />
<rect width="1" height="2" x="2" fill="#d2232c" />
</svg>
